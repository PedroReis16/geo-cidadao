# geo-cidadao

Antes de tentar realizar a execução do projeto, tenha certeza que possui a biblioteca NPM instalada em sua máquina.

### Inicializando projeto da UI

1. A partir da pasta base do projeto, digite o seguinte comando no terminal para acessar a pasta específica da interface:

```env
    cd geo-cidadao-ui
```

2. Agora, a partir do diretório da aplicação, caso nunca tenha executado o projeto, execute o seguinte comando no terminal para instalar as dependências do projeto:

```env
    npm install
```

3. Com as dependências instaladas, basta executar o comando o seguinte comando para inicializar a interface:

```env
    npm run dev
```

A interface será inicializada na rota local **http://localhost:5173**
